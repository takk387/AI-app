# Setup Instructions

1. Ensure you have Node.js (v18+) installed.
2. Optionally install pnpm globally:

   npm install -g pnpm

3. Create `.env.local` from `.env.example` and add your OpenAI API key.
4. Install dependencies:

   pnpm install

5. Start development server:

   pnpm run dev

If you prefer npm:

   npm install
   npm run dev
