# Personal AI App Builder

Minimal scaffold for the Personal AI App Builder.

Getting started

1. Copy `.env.example` to `.env.local` and set `OPENAI_API_KEY`.
2. Install dependencies: `pnpm install` (or `npm install`).
3. Run the dev server: `pnpm run dev` (or `npm run dev`).

This repository is a small scaffold generated from the provided download manifest.
